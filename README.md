# sdf-cli
sdf-cli

一个快速构建sdf vue基础目录结构的脚手架工具

# 使用

Install

`npm install -g sdf-cli`

Create a new app

`sdf-cli-service my_app`

Install Dependencies

`cd my_app && npm install`

`npx sdf-cli-service definesys-cli-project`
`npx sdf-cli-service definesys-cli-project --lang ts`
